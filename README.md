# 🎬 Movie Recommendation System

This project builds a movie recommendation system using Python and Pandas by analyzing user ratings and finding similarities between movies through collaborative filtering techniques.

---

## 🧠 Project Overview
The system analyzes user-movie rating data to identify relationships between movies and generate personalized recommendations based on similarity scores.

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Google Colab  

---

## 🚀 Features
- Movie similarity detection  
- Collaborative filtering-based recommendations  
- Data cleaning and preprocessing  
- Rating analysis and transformation
